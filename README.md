# wodensdag

> Yesterday's TV

## Dev

### Setup

[Download and install Node.js.](https://nodejs.org/en/download/)

Download and unpack, or clone, this repository and open a terminal window in the directory.

### Install Dependencies

```
$ npm install
```

### Run

```
$ npm start
```

### Build App

```
$ npm run pack
```

### Build Distribution Files

```
$ npm run dist
```
