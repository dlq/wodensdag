# wodensdag

> Yesterday's TV

## Dev

### Setup

[Download and install Node.js.](https://nodejs.org/en/download/)

```
$ git clone https://github.com/dlq/wodensdag
$ cd wodensdag
```

### Install Dependencies

```
$ npm install
```

### Run

```
$ npm start
```

### Build App

```
$ npm run pack
```

### Build Distribution Files

```
$ npm run dist
```
